# SauceDemo Automation Testing Project

## Project Overview
This is a professional Automation Framework built to test the **SauceDemo** web application. It covers the end-to-end Login functionality with various test scenarios.

## Tech Stack
**Language:** Java
**Automation Tool:** Selenium WebDriver
**Framework:** TestNG
**Design Pattern:** Page Object Model (POM)
**Build Tool:** Maven

## Scenarios Covered (7 Test Cases)
- [x] Successful Login with standard user.
- [x] Login with Invalid Password (Assertion on Error Message).
- [x] Login with Locked Out User.
